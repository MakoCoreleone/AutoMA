# AutoMA
帮你玩《二重螺旋》的自动化工具。

## 功能介绍

  1.自动60级皎皎币
  2.自动65级mod

## 食用方法

### Windows

  对于绝大部分用户，请下载 AutoMA-win-x86_64-vXXX.zip
  
  若确定自己的电脑是 arm 架构，请下载 AutoMA-win-aarch64-vXXX.zip
  
## 投喂地址
* 爱发电 https://afdian.com/a/MakoCoreleone

## 使用说明

  *目前只支持桌面端（非模拟器），且游戏需使用无边框窗口运行，通过修改系统分辨率可以降低能耗
  *前往release下载对应版本
  *建议先运行游戏，再运行脚本
  *解压后，使用管理员权限运行MFAAvalonia.exe
  *可以勾选设置闪避键进行自定义，默认为左shift键（游戏默认设置）
  *可以勾选自动战斗设置进行自定义，默认为循环按E
  *手动选择游戏窗口，然后直接运行对应任务即可，如果出现问题，先进行截图测试，如果截图失败，在设置-连接设置更改捕获方式和触控模式
  *不能携带增加移速的mod，推荐使用水母作为主控
  *游戏须在前台运行，需要后台挂机刷可以参考后台运行

## 问题反馈
推荐以Issues方式反馈问题，请一并将解压目录下“debug/maa.log”上传至附件

## 常见问题
1.使用 MaaDebugger 或 MaaPicli 时弹窗报错，应用程序错误：应用程序无法正常启动( **0xc0000142** )

<img alt="LOGO" src="https://github.com/MakoCoreleone/AutoMA/blob/main/error142.png"/>

一般是电脑缺少某些运行库，请安装一下 [vc_redist](https://aka.ms/vs/17/release/vc_redist.x64.exe)
